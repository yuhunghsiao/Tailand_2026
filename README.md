# Thailand 2026 Family Trip 🇹🇭

歡迎來到 **2026 泰國家族旅行** 專案！
這是一個專為家族旅遊設計的行程資訊網頁，方便大家隨時查看行程、住宿與相關分帳費用。

## 📅 旅遊日期
**2026年 4月 2日 (四) - 4月 8日 (三)**，共 7 天 6 夜。

## 📍 主要行程亮點
*   **在地體驗**：美功鐵道市場、空叻瑪榮水上市場 (Khlong Lat Mayom)、吞武里海鮮市場。
*   **曼谷經典**：大皇宮、臥佛寺、鄭王廟 (Wat Arun)、四面佛參拜。
*   **熱鬧市集**：洽圖洽週末市場 (Chatuchak)、唐人街 (Yaowarat)。
*   **舒適住宿**：入住「曼谷茉莉花豪華公寓 (Jasmine Grande Residence)」，享受寬敞的雙臥室套房空間。

## 🛠️ 技術說明
本網頁使用純 HTML5 + Tailwind CSS (CDN) 建置，設計風格專為手機與平版瀏覽設計 (Responsive Design)。無需複雜環境建置。

### 檔案結構
*   `index.html`: 主要行程與分帳網頁。
*   `README.md`: 專案簡介說明。
*   `.gitignore`: 系統隱藏檔過濾清單。

## 🚀 如何瀏覽
由於採用純前端技術，您只需在電腦或手機上下載本專案，並直接透過瀏覽器 (Chrome, Edge, Safari, iOS Safari 等) 點擊開啟 `index.html`，即可順暢瀏覽完整內容。
