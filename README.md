# Thailand 2026 Family Trip 🇹🇭

歡迎來到 **2026 泰國家族快樂行** 專案！
這是一個專為家族旅遊設計的行程資訊網頁，方便大家隨時查看行程、住宿與相關費用。

## 📅 旅遊日期
**2026年 2月 13日 (五) - 2月 20日 (五)**，共 8 天 7 夜。

## 📍 主要行程亮點
*   **曼谷經典**：大皇宮、臥佛寺、鄭王廟 (泰服體驗)。
*   **在地體驗**：丹嫩沙朵水上市場、美功鐵道市場、吞武里海鮮市場。
*   **新年氣氛**：唐人街 (耀華力路) 走春、感受農曆新年熱鬧氛圍。
*   **親子同樂**：洽圖洽週末市場、SEA LIFE 海洋世界。

## 🛠️ 技術說明
本網頁使用純 HTML5 + Tailwind CSS (CDN) 建置，無需複雜建置流程，直接開啟 `index.html` 即可瀏覽。

### 檔案結構
*   `index.html`: 主要行程網頁。
*   `.gitignore`: 忽略不必要的系統檔案。

## 🚀 如何瀏覽
1.  下載此專案。
2.  使用瀏覽器 (Chrome, Edge, Safari 等) 開啟 `index.html` 檔案。
